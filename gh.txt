GitHub:
- GitHub - это веб-платформа для хостинга и совместной разработки программного обеспечения.
- Позволяет разработчикам хранить, управлять и отслеживать версии своего кода с использованием системы контроля версий Git.

Основные команды Git:
1. `git init`: Инициализация нового репозитория.
2. `git clone <URL>`: Клонирование удаленного репозитория на локальную машину.
3. `git add <файлы>`: Добавление изменений в индекс.
4. `git commit -m "Сообщение коммита"`: Создание коммита с фиксацией изменений.
5. `git pull`: Получение изменений с удаленного репозитория.
6. `git push`: Отправка изменений на удаленный репозиторий.
7. `git branch`: Просмотр списка веток.
8. `git checkout <ветка>`: Переключение на другую ветку.
9. `git merge <ветка>`: Слияние изменений из выбранной ветки в текущую.
10. `git log`: Просмотр истории коммитов.
11. `git status`: Просмотр статуса файлов в репозитории.
12. `git reset <файл>`: Отмена индексации файла.
13. `git stash`: Скрытие изменений для временной работы на другой ветке.
14. `git remote`: Просмотр списка удаленных репозиториев.

GitHub команды:
1. `git remote add origin <URL>`: Привязка локального репозитория к удаленному на GitHub.
2. `git push -u origin <ветка>`: Отправка локальных изменений на GitHub и установка отслеживания.
3. `git pull origin <ветка>`: Получение изменений с GitHub.
4. `git clone <URL>`: Клонирование удаленного репозитория с GitHub на локальную машину.
5. `git fork`: Создание копии чужого репозитория на GitHub.

GitHub предоставляет возможность совместной работы над проектами, отслеживания исходного кода, 
управления задачами и многими другими функциями, упрощая процесс разработки программного обеспечения.
